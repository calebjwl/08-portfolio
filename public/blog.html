The box model review assignment was refreshing in two ways. First, the fundamentals of the assignment was loosely based on concepts and practices we had already learned in day two. Everything I had learned and practiced this day further solidified concepts I had already known as well as those I was still unsure about. The position layout assignment taught me not only how to position fixed elements on a page, but also how to better break down sections of code when considering the end goal for whichever website design I wanted to create.
